# Product Overview

SJ Gestor is a comprehensive business management platform that combines billing, client management, and WhatsApp automation for Brazilian businesses.

## Core Features

- **Client Management**: Store client information with recurring billing capabilities
- **Product & Service Catalog**: Manage products/services with pricing
- **Charge Generation**: Create and track charges linked to clients and products
- **PIX Integration**: Brazilian instant payment system integration via Mercado Pago
- **WhatsApp Automation**: Automated messaging using Baileys library for payment reminders and notifications
- **Message Templates**: Customizable message templates for different scenarios
- **Subscription Plans**: Multi-tier subscription system for users
- **Dashboard Analytics**: Business insights and reporting

## Target Market

Brazilian small to medium businesses needing integrated billing and communication solutions, particularly those using PIX payments and WhatsApp for customer communication.

## Key Integrations

- Mercado Pago (payment processing)
- WhatsApp Business API (via Baileys)
- PIX payment system
- PostgreSQL database
